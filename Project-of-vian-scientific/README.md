## How to Use This Tool

1.  **Zip your project:** Create a `.zip` file of the source code you want to analyze.
2.  **Upload:** Use the "Upload Project ZIP" button on the main screen.
3.  **Analyze & Chat:** The tool will process your files. Once ready, you can ask questions about your project's code, structure, dependencies, and more.
